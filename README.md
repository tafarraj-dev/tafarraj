# Tafarraj 🎬

A curated Arabic-language platform for discovering Asian dramas (Korean, Chinese, Japanese, Thai, Turkish) — search, browse by country, and find verified watch links across multiple streaming sources.

## Features
- Browse dramas by country: Korean, Chinese, Japanese, Thai, Turkish
- Filter by status (ongoing/completed) and genre
- Multiple watch-link sources per drama with info on subtitles, ads level, and episode completeness
- Arabic-first UI with full RTL support
- User accounts with personal watchlists

## Tech Stack
- **Backend:** Django (Python)
- **Database:** PostgreSQL (production), SQLite (local dev)
- **Static files:** WhiteNoise
- **Deployment:** Render
- **Media storage:** Cloudinary
